🌍 NeoScript Stable Release 1.0

🔥 Update Stable Release 1.0 !!!

This update is a breakthrough with 5 new features:
- New note command: Add comments to your code using plain text without affecting execution.
- Smart write command: Now supports outputting both raw text and variable values seamlessly.
- New input_str command: Interact with your programs by entering string/text data from the keyboard.
- Enhanced Variable Engine: Automatically detects and handles both numeric and string variables.
- Optimized Math Logic: The add and sub commands now support variable-to-variable operations.

"Made for human!" NeoScript is a lightweight programming language. Goal: Make coding as easy as writing English. No ; or {} required.

📜 Current Commands:

- write "message": Output text or variable values to the console (Enhanced!).
- input "variable": Prompt user for numeric input.
- input_str "variable": Prompt user for string/text input (New!).
- set "name" value: Initialize a numeric or string variable (Updated!).
- add "name" value/variable: Add a value or another variable to a variable (Enhanced!).
- sub "name" value/variable: Subtract a value or another variable from a variable (Enhanced!).
- wait X: Pause program execution for X seconds.
- clear: Clear the entire console window for a fresh workspace.
- color "name": Change the text color (e.g., color "cyan", "green", "red").
- note "message": Add non-executable notes/comments to your script (New!).

Python 3.11 or higher (Required to run the interpreter).

🚀 How to Use (4 Steps)

1. Download: Download the source files and extract them to your project folder.
2. Setup: Make sure you have Python 3.11 or higher installed and added to your PATH.
3. Code: Open ide.ns with any text editor and write command you want to execute in it.
4. Run: Double-click run.bat to see your code in action.

📜 File Structure

- Neoscript_SR_v1.0.py: The core interpreter (the "brain") of the language.
- ide.ns: Your script file where you write your NeoScript code.
- run.bat: The quick-launch file for easy execution on your local machine.
- run_for_others.bat: Standard launch file for sharing with friends.
